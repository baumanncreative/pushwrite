# Code of Conduct

This project follows the Contributor Covenant Code of Conduct 2.1 in a shortened form.

## Our Standard

- Be respectful, direct, and constructive.
- Assume good intent, but focus feedback on behavior and technical outcomes.
- Welcome contributors with different backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination
- Personal attacks or insulting language
- Publishing private information without permission
- Disruptive behavior that prevents productive collaboration

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that do not follow this Code of Conduct. Report serious concerns privately to the repository maintainers.

Adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
